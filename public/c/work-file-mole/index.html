どんなキャラ？

大量の書類ファイルの中から顔を出すもぐら妖怪。目立たない場所で情報を整理し、チームの土台を支える。

このタイプの特徴

前に出るより、周りを支えることで価値を発揮するタイプです。人の抜け漏れに気づいたり、困っている人を自然に助けたりできる縁の下の力持ちです。

向いている働き方：アシスタント・秘書・カスタマーサポート・チーム補佐・調整役

注意ポイント：支えすぎると、便利な人として扱われることがあります。自分の担当範囲と限界を言葉にすることも仕事の一部です。
